# Smart Library Request Workflow (ServiceNow)

This project implements a Smart Library Request system in ServiceNow using:

* Service Catalog
* Flow Designer
* Business Rules
* Script Includes

## Features

* Automated request routing
* Smart approval system
* Book availability check
* Task creation for fulfillment
* Notifications

## Workflow

1. User submits request
2. System checks availability
3. Auto-approval or manager approval
4. Task assignment
5. User notified

## Requirements

* ServiceNow instance (Rome+)
* Flow Designer enabled

🧠 Script Include – Availability Check

📁 scripts/checkAvailability.js

var LibraryUtils = Class.create();
LibraryUtils.prototype = {
    initialize: function() {},

    checkAvailability: function(title) {
        var gr = new GlideRecord('cmdb_ci_book');
        gr.addQuery('name', title);
        gr.query();

        if (gr.next()) {
            return true;
        }
        return false;
    },

    type: 'LibraryUtils'
};
🤖 Auto Approval Logic

📁 scripts/autoApproval.js

function autoApprove(request) {
    if (request.cost < 1000) {
        return true;
    }
    return false;
}
🔀 Routing Logic

📁 scripts/routingLogic.js

function routeRequest(type) {
    switch(type) {
        case 'Book':
            return 'Librarian Team';
        case 'Journal':
            return 'Research Team';
        case 'eBook':
            return 'Digital Access Team';
        default:
            return 'General Library Team';
    }
}
⚡ Business Rule – Set Priority

📁 business_rules/setPriority.js

(function executeRule(current, previous) {

    if (current.variables.urgency == 'High') {
        current.priority = 1;
    } else if (current.variables.urgency == 'Medium') {
        current.priority = 2;
    } else {
        current.priority = 3;
    }

})(current, previous);
